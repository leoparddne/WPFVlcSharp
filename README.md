# WPFVlcSharp

#### 介绍
使用VLCSharp编写的接触播放器界面

测试代码时记得调整测试视频的路径(默认为string url = "C:\\Users\\ives\\Desktop\\zhaohuo.mp4")

![界面效果](mainwindow-1.png)

#### 软件架构
WPF
VLCSharp
.net6
handycontrol


#### 使用说明

1.  支持点击界面移动窗口
2.  支持全屏、ESC退出全屏
3.  暂停、播放
4.  静音
5.  支持点击进度条位置&拖动进度条位置
